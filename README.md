# ruby-bigwig

Ruby wrapper to [libBigWig](https://github.com/dpryan79/libBigWig).

## Installation

```
git clone --recursive https://github.com/kojix2/bw-ruby
cd bw-ruby
bundle install
bundle exec rake libbigwig:build
bundle exec rake install
```
